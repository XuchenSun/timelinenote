Política de Privacidade

Última atualização: 24 de abril de 2026

TimelineNote é desenvolvida por Xuchen Sun. Esta política de privacidade explica como lidamos com seus dados.

Armazenamento de Dados

Todas as notas, categorias, tags e configurações são armazenadas localmente no seu dispositivo. Não temos acesso aos seus dados.

Sincronização com iCloud

Se ativada, os dados são sincronizados via Apple CloudKit. Não podemos acessar seus dados do iCloud.

Recursos de IA (Opcional)

Os recursos de IA opcionais exigem sua própria chave de API do Moonshot AI (Kimi). O conteúdo é enviado para api.moonshot.cn para processamento. Sua chave de API é armazenada localmente. Não a coletamos nem acessamos.

Reconhecimento de Voz

A conversão de voz em texto usa o framework Speech da Apple no dispositivo. O áudio não é enviado externamente.

Câmera e Biblioteca de Fotos

Usadas para anexar imagens às notas. As fotos são armazenadas localmente. Não as enviamos nem compartilhamos.

Análise e Rastreamento

Não usamos análises, anúncios ou rastreamento. Nenhum dado de uso ou informação pessoal é coletado.
