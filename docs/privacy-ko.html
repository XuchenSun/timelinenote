개인정보 처리방침

최종 업데이트: 2026년 4월 24일

TimelineNote는 Xuchen Sun이 개발했습니다. 이 개인정보 처리방침은 귀하의 데이터를 어떻게 처리하는지 설명합니다.

데이터 저장

모든 노트, 카테고리, 태그 및 설정은 기기에 로컬로 저장됩니다. 저희는 귀하의 데이터에 접근할 수 없습니다.

iCloud 동기화

활성화된 경우 Apple CloudKit을 통해 데이터가 동기화됩니다. 저희는 귀하의 iCloud 데이터에 접근할 수 없습니다.

AI 기능 (선택 사항)

선택적 AI 기능을 사용하려면 귀하 소유의 Moonshot AI (Kimi) API 키가 필요합니다. 콘텐츠는 처리를 위해 api.moonshot.cn으로 전송됩니다. API 키는 로컬에 저장되며, 저희는 이를 수집하거나 접근하지 않습니다.

음성 인식

음성-텍스트 변환은 Apple의 온디바이스 Speech 프레임워크를 사용합니다. 오디오는 외부로 전송되지 않습니다.

카메라 및 사진 보관함

노트에 이미지를 첨부하는 데 사용됩니다. 사진은 로컬에 저장되며 업로드하거나 공유하지 않습니다.

분석 및 추적

분석, 광고, 추적 기능을 일절 사용하지 않습니다. 사용 데이터나 개인 정보를 수집하지 않습니다.
