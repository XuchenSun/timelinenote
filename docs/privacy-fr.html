Politique de confidentialité

Dernière mise à jour : 24 avril 2026

TimelineNote est développée par Xuchen Sun. Cette politique de confidentialité explique comment nous traitons vos données.

Stockage des données

Toutes les notes, catégories, étiquettes et paramètres sont stockés localement sur votre appareil. Nous n'avons pas accès à vos données.

Synchronisation iCloud

Si activée, les données se synchronisent via Apple CloudKit. Nous ne pouvons pas accéder à vos données iCloud.

Fonctionnalités IA (Optionnelles)

Les fonctionnalités IA optionnelles nécessitent votre propre clé API Moonshot AI (Kimi). Le contenu est envoyé à api.moonshot.cn pour traitement. Votre clé API est stockée localement. Nous ne la collectons pas et n'y accédons pas.

Reconnaissance vocale

La conversion parole-texte utilise le framework Speech d'Apple sur l'appareil. L'audio n'est pas transmis à l'extérieur.

Appareil photo et photothèque

Utilisés pour joindre des images aux notes. Les photos sont stockées localement. Nous ne les téléchargeons ni ne les partageons.

Analyses et suivi

Nous n'utilisons aucune analyse, publicité ou suivi. Aucune donnée d'utilisation ni information personnelle n'est collectée.
