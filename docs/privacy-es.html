Política de Privacidad

Última actualización: 24 de abril de 2026

TimelineNote está desarrollada por Xuchen Sun. Esta política de privacidad explica cómo gestionamos tus datos.

Almacenamiento de Datos

Todas las notas, categorías, etiquetas y ajustes se almacenan localmente en tu dispositivo. No tenemos acceso a tus datos.

Sincronización con iCloud

Si está activada, los datos se sincronizan a través de Apple CloudKit. No podemos acceder a tus datos de iCloud.

Funciones de IA (Opcional)

Las funciones de IA opcionales requieren tu propia clave de API de Moonshot AI (Kimi). El contenido se envía a api.moonshot.cn para su procesamiento. Tu clave de API se almacena localmente. No la recopilamos ni accedemos a ella.

Reconocimiento de Voz

La conversión de voz a texto utiliza el framework Speech de Apple en el dispositivo. El audio no se envía al exterior.

Cámara y Fototeca

Se utilizan para adjuntar imágenes a las notas. Las fotos se almacenan localmente. No las subimos ni compartimos.

Análisis y Seguimiento

No utilizamos análisis, publicidad ni seguimiento. No se recopilan datos de uso ni información personal.
